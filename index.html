Research Approach

This project evaluates ChatGPT's efficiency by leveraging advanced web technologies. Our setup integrates powerful tools such as:

  • MongoDB: For data storage and management.
  • Node.js: For server-side scripting and API handling.
  • ChatGPT API: For evaluating ChatGPT's response quality in various scenarios.

These technologies allow us to design a robust, data-driven evaluation process that tests ChatGPT’s ability to handle diverse queries and contexts effectively.

Dataset Overview

The dataset used in this evaluation consists of a diverse set of questions spanning three key domains:

  • History: To test factual recall and reasoning about past events.
  • Social Science: To evaluate ChatGPT's understanding of societal concepts and theories.
  • Computer Security: To assess technical expertise in a specialized field.

Each domain was selected to ensure a comprehensive analysis of ChatGPT’s strengths and limitations, providing insights into its performance across both general and technical areas.

Results

The results section will present visualizations and metrics showcasing ChatGPT's performance in each domain. These charts and graphs will highlight areas of strength, improvement opportunities, and comparative accuracy across the selected fields.
